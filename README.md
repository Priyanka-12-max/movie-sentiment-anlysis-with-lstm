# movie-sentiment-anlysis-with-lstm

Project Objective

The objective of this project is to analyze movie reviews and classify their sentiment as positive or negative using a Long Short-Term Memory (LSTM) neural network. The model learns patterns from textual data to understand emotions and opinions expressed in reviews

Dataset
Dataset used: IMDB Movie Reviews Dataset
Total reviews: 50,000

Classes:
Positive sentiment
Negative sentiment

The dataset is split into training and testing sets.
